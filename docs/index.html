What it does

# Ingest and grade a session $ sesh log ~/.claude/projects/myapp/*.jsonl [A-] session-2026-03-13.jsonl — 47 calls, 1 error, 0 patterns # See what happened $ sesh reflect --last 1 Session: session-2026-03-13 Grade: A- (91/100) Model: claude-opus-4-6 Duration: 34 min · 47 tool calls · 1 error Patterns: ⚠ missed_parallelism 2 independent reads done sequentially Deductions: -2 error rate (2.1%) -5 missed parallelism (2 cases) -2 low read ratio (0.45) Bonuses: +3 high read ratio bonus (reads before writes) +2 zero blind edits

Features

Grading

0-100 score with A+ to F grades. Deductions for anti-patterns, bonuses for good habits.

Trend analysis

Track improvement or regression across sessions. Spot recurring patterns.

Auto-discovery

sesh watch monitors transcript directories and auto-ingests new sessions.

Full-text search

Search past sessions by tool calls, patterns, or raw text.

Handoff docs

Generate context handoff documents for session continuity.

Zero dependencies

Pure Python stdlib. No external packages required for core functionality.
