How it works

Auto-discovery

sesh analyze finds your most recent session automatically. No file paths needed.

Closed-loop feedback

--feedback writes findings to CLAUDE.md. Each run replaces the last. Agents improve.

Zero dependencies

Pure Python stdlib. No external packages. Installs in seconds.

Machine-readable

--json on every command. Exit codes for CI. Built for automation.

Session replay

Step-by-step playback with pattern annotations. See exactly what happened.

Prompt debugging

Search agent thinking blocks. Find the decision point behind any action.
